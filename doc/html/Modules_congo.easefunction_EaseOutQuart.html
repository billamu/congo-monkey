Detailed Discussion

Ease-out, quartic (decelerates to zero). Changes faster than cubic.

Method Documentation

Method GetValue : Float ( t:Float )

Returns a value between 0.0 and 1.0 for the given progress value t. t should be in range 0.0 to 1.0. If t => 1.0, will return 1.0.
