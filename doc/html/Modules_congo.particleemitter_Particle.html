Class Particle

Simple data structure to hold particle state (all fields are public for simplicity).

Fields:
  • alpha : Float
  • life : Int
  • posX : Float
  • posY : Float
  • rotation : Float
  • scale : Float
  • velX : Float
  • velY : Float

Field Documentation

Field alpha : Float

Alpha (opacity). Default 1.0.

Field life : Int

Lifetime of particle remaining, in ms units. Default 5000.

Field posX : Float

X position.

Field posY : Float

Y position.

Field rotation : Float

Rotation.

Field scale : Float

Scale. Default 1.0. Individual particle scales will combine with this overall scale.

Field velX : Float

X velocity.

Field velY : Float

Y velocity.
