Detailed Discussion

A sprite with built-in physics behaviour. Requires box2d module. See also: PhysicsWorld class.

** THIS CLASS IS UNDER DEVELOPMENT ** - it is useable but has limited functionality.

Field Documentation

Field doRotateSprite : Bool

Set this to False to prevent the Sprite rotating to match the underlying box2d body.

Field m_body : b2Body

The box2d body.

Field objName : String

User-defined name. See also objType. You can use this field to label different game objects (helpful when casting etc).

Field objType : Int

User-defined type. You can use this field to label different game objects (helpful when casting etc).

Method Documentation

Method AddVelocity : Void ( vel:b2Vec2 )

Gives the body a kick of linear velocity in the specified direction. Will wake-up the body if it was sleeping.

Method Update : Void ( dT:Int )

Main Update function. The Sprite is moved and (optionally) rotated to match the underlying box2d body. Derived classes can override this if required.
