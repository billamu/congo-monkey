Detailed Discussion

Main sprite class. Can load atlas/spritesheets, import from TexturePacker. Since it is derived from DisplayItem, it can own child items and run actions.

Uses TextureCache to store it's Images.

See also AnimatedSprite, for frame-based animation.

Constructor Documentation

Method New ()

Default constr. No images are added, but can be added later using SetMainImage.

Method New ( myImg:Image )

Creates a Sprite with a single image. See also CongoImageLoader (in DisplayUtils) to load an Image. You can set Image.MidHandle on the image as required.

Method New ( tpimg:Image, dataName:String, imageName:String )

Creates a Sprite from a Texturepacker file. See LoadFromTP for more information.

Sets the Bounding box height scale factor, default 1.0. Used by CollisionLayer.

Method BoundingBoxWidthScale : Float () Property

Returns the Bounding box width scale factor, default 1.0. Used by CollisionLayer.

Method BoundingBoxWidthScale : Void ( scale:Float ) Property

Sets the Bounding box width scale factor, default 1.0. Used by CollisionLayer.

Method CollisionsDisabled : Bool () Property

Returns the status of the collision detection flag for this Sprite. Used in CollisionLayer (or use for your own purposes).

Method DisableCollisions : Void ( state:Bool=true ) Property

Set this flag to remove the Sprite from any collision detection performed in a CollisionLayer (or use for your own purposes).

Method ImageHeight : Float () Property

Returns the current Image height (original, non-scaled), or 0 if no current image.

Method ImageWidth : Float () Property

Returns the current Image width (original, non-scaled), or 0 if no current image.

Method TransformationMatrix : Float[] () Property

For advanced use. Returns the Sprite's current transform matrix (6 float array). You may need this for custom Draw code. Note, this data only gets set in the Draw step.

Method Documentation

Method BoundingBoxHeight : Float ()

Returns the bounding box height. Used by CollisionLayer. By default this is the same as the current ImageHeight (adjusted for sd/hd/xhd res scalefactor), and also scaled by the user-defined boundingBoxHeightScale. Local, not global coords.

Method BoundingBoxWidth : Float ()

Returns the bounding box width. Used by CollisionLayer. By default this is the same as the current ImageHeight (adjusted for sd/hd/xhd res scalefactor), and also scaled by the user-defined boundingBoxHeightScale. Local, not global coords.

Method DrawBoundingRect : Void ()

For debugging purposes, draws the bounding rect (local coords). Color is based on bounding rect size.

Method DrawGlobalBoundingRect : Void ()

Experimental, not for use yet. Draws the bounding rect using the cached global transform. Color is based on bounding rect size.

Method GetBoundingRect : Rect ()

Returns the bounding rect in local coordinates. Used by CollisionLayer hit detection. The box size can be customised (e.g. reduced or expanded beyond usual Image size) by overriding BoundingBoxHeight or BoundingBoxWidth. Known issues: doesn't deal with non-centre rotation handles.

Method GetCurrentImage : Image ()

Returns the current main Image (could be Null if not set).

Method InitResScaler : Void ( autoDetect:Bool=true, scale:Float=1.0 )

Internal. Sets res scale factor based on display settings. i.e. if the display is set-up for hd images then the res scale will be set to 2.0 to double the base size. This is called automatically from the constr, or when the main image is changed, you don't usually need to do this manually.

Method LoadFromTP : Bool ( tpimg:Image, dataName:String, imageName:String )

Loads Image from Texturepacker file (Generic xml format). Can be called on a current Sprite, or more usually, called from the constructor on creation of the Sprite.

Will throw exception if image atlas is invalid. This method uses TextureCache for future requests for the same image resource.

Important! Do NOT use Image.MidHandle on the sprite sheet, the frames will be centered automatically.

Note, don't supply 'hd' or other extension names for the xml or the image names, this routine will add them. All names are case-sensitive.

Method PaintItem : Void ()

All DisplayItems must implement the base class PaintItem method.

Method SetMainImage : Void ( img:Image )

Manually change the Image associated with the Sprite.

Method Update : Void ( dT:Int )

Main Update. Derived classes can override this, remember to call Super.Update()!
