Detailed Discussion

Action classes, for sprite move/scale/fade tweens. Various ease functions are supported. Multiple Actions can be run simultaneously, or sequences can be set up via ActionSequence.

All timing is in milliseconds. Completed actions trigger the ActionCompleted 'callback' in DisplayItem.

Actions can generally be applied to any DisplayItem object, which includes layers etc.
