Function OpenBrowserURL : Void ( url:String, windowName:String="_blank" )

DEPRECATE? - use mojo OpenUrl instead. However, this function can open in a new window. [HTML5] Opens a browser page with the given URL. Exact behaviour depends on target platform. Windowname is usually one of _blank, _self, _parent, _top.
