Detailed Discussion

A particle emitter system. Derives from Sprite, can be treated as such, added as a child to another sprite or layer. (main data is public for simplicity).

If you use the built-in particle texture sheet, remember to copy congoparticles.png into your data folder.

Field Documentation

Field blendMode : Int

Standard Monkey blend mode, currently AdditiveBlend or AlphaBlend but also depends on target.

Field continuous : Bool

Set to True to keep emitting until emitter lifetime elapsed.

Field emitRate : Float

Defines the delay between each particle being emitted, in ms. Default 100 ms. Only applies if continuous emitter.

Field fadeRate : Float

Fade rate.

Field gravity : Point

Gravity vector. Default to slow y fall.

Field lifeTime : Int

Particle lifetime in ms. Default 5000 ms.

Field lifeVariance : Int

Particle lifetime variance. Default 1000 ms.

Field particles : List<Particle>

Stores all our particles.

Field radialAccel : Float

Radial acceleration, default 0.0. Note - gravity and radialAccel dont really work together, use one or the other. Radial accel is more cpu intensive (has to normalise a direction vector)

Field shrinkRate : Float

Shink rate. Can be negative to grow in size.

Field spinRate : Float

Spin rate of particles. Will spin in direction of initial rotation. Hence, must have a non-zero start angle for this to work.

Field startAlpha : Float

Start alpha (opacity). Default 1.0

Field startAlphaVariance : Float

Alpha variance.

Field startAngle : Float

Start rotation angle.

Field startAngleVariance : Float

Start angle variance.

Field startPosXVariance : Float

Start X position variance.

Field startPosYVariance : Float

Start Y position variance.

Field startSize : Float

Starting size (scale).

Field startSizeVariance : Float

Starting size variance.

Field startVelX : Float

Start X velocity.

Field startVelXVariance : Float

Start X velocity variance.

Field startVelY : Float

Start Y velocity.

Field startVelYVariance : Float

Start Y velocity variance.

Field texture : Image

Particle texture. Currently, all particles share the same texture.

Field timer : Timer

Main timer for the particle system.

Constructor Documentation

Method New ( particleId:Int=0, numParticles:Int=20 )

Creates an emitter using a built-in texture, numbered 0 to 15. See the "data/congoparticles.png" texture sheet. Move the emitter using the standard Position parameters. Use lifetime -1 for continuous emit. Set other parameters individually, or use one of the default sets.

Tips: try using a small number of particles, e.g. 20-100. Additive blending can help with the effect. Note, black particles will not be visible if blend mode is AdditiveBlending.

Method New ( texture:Image, numParticles:Int=20 )

As above, but using a custom particle texture. See congoparticles for example texure size.
