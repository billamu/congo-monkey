Detailed Discussion

A multi-line text label. It wraps angelfont classes to provide a Sprite interface. Labels cannot have child sprites.

Constructor Documentation

Method New ( text:String, font:AngelFont, xpos:Float=0, ypos:Float=0, align:Int=angelfont.align_left, boxWidth:Int=1000 )

Creates a text label. Remember to create the font first (use CongoFontLoader) -- Labels can share fonts. Angelfont alignment options are ALIGN_LEFT, ALIGN_CENTER and ALIGN_RIGHT. Useful characters: ~n = newline, ~q = double quotation mark (see Monkey docs)
